# NOTEMIND-AI
NOTEMIND-AI is an AI-powered note organisation platform built to turn messy, text-heavy PDFs into clean, readable, and well-structured notes.
Instead of manually reading through long documents, you can simply upload a typed PDF and let NOTEMIND-AI extract the important content for you — saving time and mental effort.

----------------------------- What does it do------------------------------------

# Upload typed PDF files
# Extract and process text automatically
# Convert PDFs into clean, structured notes
# Fast, simple, and easy to use interface
# Helps you stay organised and focused

Perfect for students, researchers, and anyone dealing with lots of documents.
-----------------------------Tech Stack------------------------------------------

Backend: Python (Flask)
Frontend: HTML, CSS, JavaScript
PDF Processing: Python PDF libraries

--------------------------- How to run locally------------------------------------

Clone the repository:

git clone https://github.com/mdzeeshan767/NOTEMIND-AI.git
Navigate to the project folder:
cd NOTEMIND-AI
Run the backend:
python backend/app.py
Open your browser and go to:
http://127.0.0.1:5000/ui
